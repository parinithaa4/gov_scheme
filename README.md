# 🎨 Frontend -- Smart Welfare Scheme Recommendation System

## 📌 Overview

This is the frontend of the Smart Welfare Scheme Recommendation System.
It focuses on providing a smooth, interactive, and accessible user
experience for collecting user data and displaying personalized
government scheme recommendations.

The interface is designed to feel conversational and simple, especially
for users who may not be comfortable with traditional complex forms.


## ⚙️ Tech Stack

-   **Framework:** React.js\
-   **Styling:** Custom CSS (Glassmorphism-inspired UI)\
-   **State Management:** React Hooks\
-   **API Communication:** Fetch (`api/client.js`)

## 🌍 Features
